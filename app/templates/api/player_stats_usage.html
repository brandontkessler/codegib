Using the NBA Stats API

How to Use

Using the API requires querying with a "player" parameter:

This must include first name, last name, and (position) OR in some cases, first initial, last name, and (position)

Examples

Example: LeBron James

Relative Path: '/api/nba_stats?player=LeBron James (F)'

Example: Michael Kidd-Gilchrist

Relative Path: '/api/nba_stats?player=M. Kidd-Gilchrist (SF)'

Using the NBA Player Info API

How to Use

Using the API requires querying with a "player" parameter:

This must include first and last name

Examples

Example: Giannis Antetokounmpo

Relative Path: '/api/nba_player_info?player=Giannis Antetokounmpo'

Example: Frank Mason III

Relative Path: '/api/nba_player_info?player=Frank Mason III'
